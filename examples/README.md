# AumOS SDK Examples

End-to-end quickstart examples for each supported SDK language.

## Python

```bash
cd python
pip install aumos-sdk
AUMOS_API_KEY=sk-aumos-... python quickstart.py
```

## TypeScript

```bash
cd typescript
npm install @aumos/sdk tsx
AUMOS_API_KEY=sk-aumos-... npx tsx quickstart.ts
```

All examples follow the same narrative:
1. Health check
2. Agent creation with tools
3. Run execution with polling
4. Model listing
5. Governance audit logs
6. Structured error handling
7. Resource cleanup
